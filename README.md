Current features (12/13/2025):
* Sword attacks (light and heavy)
* Sprint system with stamina and stamina bar UI
* Deal damage to enemies
* Player character health system with health bar UI and death
* Ability to receive damage
* Interact with named NPCs
* Branching dialogue system with choices

First-person action role-playing game in Unreal Engine 5.5. Primarily inspired by The Elder Scrolls IV: Oblivion as well as The Elder Scrolls V: Skyrim. Learning Unreal for the first time, and this is my first project.
