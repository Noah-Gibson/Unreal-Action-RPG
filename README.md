# Untitled
First-person action role-playing game in Unreal Engine. Primarily inspired by The Elder Scrolls IV: Oblivion, also The Elder Scrolls V: Skyrim.

## Setup Instructions
1. Clone this repository
2. Install Unreal Engine 5.5
3. Download required Marketplace assets (see below)
4. Right-click .uproject â†’ "Generate Visual Studio project files"
5. Open .sln and build
6. Launch project
